# Arduino
I want to show you in this repo how to control 7-Segment-Display

What you need:
  * Arduino Duemilanove
  * 7-Segment-Display
  * Dual 7-Segment-Display
  * Push Buttons
  * Potentiometer
  * 1 x 74HC595
  * breadboard
  * wire
